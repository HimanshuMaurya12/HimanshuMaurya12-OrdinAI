# OrdinAI – AI-Powered Regulatory Intelligence Engine

OrdinAI is an advanced AI-driven regulatory intelligence platform designed to ingest, process, embed, and retrieve financial regulatory documents (RBI, SEBI, etc.) using a knowledge graph + vector search architecture.

---

##  Features

- 📄 Automated document ingestion (PDF + HTML)
- 🧠 Parent-child semantic chunking
- 🔎 High-accuracy embedding using BGE-Large
- ☁️ Qdrant Cloud vector storage
- 🕸 Knowledge Graph visualization
- ⚡ GPU-optimized ingestion pipeline
- 🧾 AI-based compliance report generation

---

## 🏗 Architecture

Frontend: React  
Backend: FastAPI  
Vector DB: Qdrant Cloud  
Embeddings: BAAI/bge-large-en-v1.5  
Parsing: Unstructured.io
